"git add ." - Добалвяет все файлы для остлеживания изменений
"git commit -m "text" " = записывает все изменения
"git push ssh-link (ссылка ssh)  master"  = закидывает в гитхаб изменения. 




"--help" - Пишется перед командой. выводит описание функций команды.
"Ls" - показывает список папок и файлов текущей директории. 
"cd" - change direction (поменять директорию) cd Desktop/ перемещает в директорию Рабочего стола. 
"cd .." - возвращает на одну папку назад. 
"mkdir" - make direction (создаёт папку-директорию) + название папки 
"touch" - создаёт файл + название файла. 
"tree ." - показывает дерево директорий. (нужна предустановка)
"mv" - Move, переносит папку. 
"code" - открывает visual studio
"cd ../../../" - в данном случае "/" выступает как разделитель предыдущих папок (..)
"cd -" - возвращает на 1 команду назад (аналог с браузером) 
"cd ." - переход в туже папку (мы никуда не переходим) 
"cd"   - переход в домашнюю папку (~)
"~"    - домашняя папка
"pwd"  - print working directory - показывает абсолютный путь до текущей папки. 
"mkdir test/new_dir" - создаёт новую директорию ("new_dir") в уже существующей папке ("test")
"rm" + "название файла" - Remove, удаление файла. 
"rm -rf" - удаление папки со всем содержимым (папки, файлы), удаляется навсегда. "rf" - Recursion Force
!!!! "sudo rm -rf /" - сносит всю ОС
"cat +название файла"  - показывает содержимое файла. 
"ls -a" - показывает все папки + скрытые папки. a=all
"ls -l" - показывает папку + свойства папки (дата создания, автор, права)
"nano +название файла" - Открывает, редактирует файл в терминале (opens and edits file), чтобы сохранить "ctr+s", чтобы выйти "ctrl+x"
